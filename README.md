# Free Return – 自由帰還軌道シミュレータ

地球低軌道（LEO）から月へ向かう自由帰還軌道をシミュレーションするWebアプリです。

## 物理モデル

- **制限三体問題（CR3BP）**: 地球–月–宇宙機の3体系を回転座標系で解く
- **数値積分**: 4次ルンゲ–クッタ法（RK4）
- **出発地点**: LEO（高度185km）
- **質量比**: μ = 0.01215（月質量 / (地球質量 + 月質量)）

## 機能

- 初速度・発射角度を自由に設定
- 回転座標系 / 慣性座標系の切り替え
- 月の重力 ON/OFF（月なしの軌道との比較）
- ズーム・パン操作
- 再生速度調整、途中停止
- WebM動画としてダウンロード

## ローカル開発

```bash
npm install
npm run dev
```

## Vercel へのデプロイ

1. このリポジトリを GitHub に push
2. [vercel.com](https://vercel.com) でアカウント作成（GitHub連携）
3. 「Add New Project」→ リポジトリを選択 → 「Deploy」
4. 設定不要で自動的にビルド＆デプロイされます

以降は `main` ブランチに push するたびに自動デプロイされます。
